# Prerequisites (or list of stuff I can't work without)

- vim 7.3
- git
- tmux
- ruby (min 1.8.7)
- rake


# How to setup (once you got the stuff above)

    cd ~
    git clone git@github.com:lukaszkorecki/DotFiles.git .DotFiles
    cd .DotFiles
    rake setup
